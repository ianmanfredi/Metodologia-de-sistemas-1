# Comandos de Ayuda y Documentación

## git help (o man git)
**Qué hace y cuándo usarlo:** Despliega el manual oficial y la documentación técnica de cualquier comando de Git. Usalo cuando no recuerdes cómo funciona un comando específico, qué parámetros acepta o qué *flags* (banderas) existen.

**Cómo usarlo:**
1. Escribí `git help <comando>` (por ejemplo, `git help log`) o `man git-<comando>` en entornos Linux/macOS.
2. Navegá por el manual usando las flechas direccionales.
3. Para buscar una palabra específica dentro del manual, escribí `/` seguido de la palabra y presioná Enter.
4. Presioná `q` para cerrar el manual y volver a la terminal.
