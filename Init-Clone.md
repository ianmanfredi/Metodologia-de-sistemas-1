# `git init`

**¿Qué hace?**

Lo que hace es iniciar un repositorio en la carpeta donde estes parado. Que significa esto? Que desde ese momento git crea una carpeta oculta llamada “.git” la cual desde ese momento va a seguir y guardar todos los cambios que se hagan.

**Ejemplo de uso:**
```bash
git init
```


# `git clone`

**¿Qué hace?**

Este comando lo que hace es copiar un repositorio existente en una carpeta de tu maquina, y no solo en ese momento empieza a guardar todos los cambios que hagas sino que te trae todo el historial de cambios que se hayan hecho desde que se inicializo el repositorio, por lo que tenes acceso a toda la historia completa del repositorio.

**Ejemplo de uso:**

```bash
git clone
```
