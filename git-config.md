# Configuracion inicial: git config
Su funcion es establecer tu identidad (nombre y correo). Esto es para que **GIT** lo adjunte a cada cambio que guardas, haciendo un historial para ver quien hizo cada modificacion.

**Ejemplo de uso:**
```bash
git config --global user.name "Ian Manfredi"
git config --global user.email "ian.manfredi12@gmail.com"
