# Trabajo Práctico: Git y GitHub - Metodología de Sistemas I

## Introducción
El presente repositorio contiene el desarrollo del Trabajo Práctico de la asignatura Metodología de Sistemas I. El objetivo principal de este proyecto colaborativo es demostrar el dominio práctico de Git y GitHub en un entorno de trabajo en equipo. A través de este repositorio, documentamos y aplicamos buenas prácticas de control de versiones, incluyendo el uso de ramas independientes, resolución de conflictos, revisión de código mediante Pull Requests y el uso de la convención de *Conventional Commits*. 

A continuación, se detalla el índice con los enlaces a la documentación técnica elaborada por el equipo, las métricas del proyecto y los recursos utilizados.
